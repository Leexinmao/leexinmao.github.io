下面是map
<c:forEach items="${map }" var="m">
	${m }
</c:forEach>
  1. items:取得需要遍历的集合
  2. var:每次遍历出来的元素
  3. varStatus:当前元素的转态 常用属性：count 取得当前记录的序号
  4. begin:取值的起始下标
  5. 取值的结尾下标
  6. step:取值的步长，表示 每隔几个取一个值
